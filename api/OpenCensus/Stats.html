Overview

The Stats module contains support for OpenCensus stats collection.

OpenCensus allows users to create typed measures, record measurements, aggregate the collected data, and export the aggregated data.

.configure ⇒ Object

Configure OpenCensus Stats. These configuration fields include parameters governing aggregation, exporting.

This configuration is also available as the stats subconfig under the main configuration OpenCensus.configure. If the OpenCensus Railtie is installed in a Rails application, the configuration object is also exposed as config.opencensus.stats.

Generally, you should configure this once at process initialization, but it can be modified at any time.

Supported fields are:

  • exporter The exporter to use. Must be an exporter, an object with an export method that takes an array of ViewData objects. See Exporters. The initial value is a OpenCensus::Stats::Exporters::Logger that logs to STDOUT.

@example:

OpenCensus::Stats.configure do |config| config.exporter = OpenCensus::Stats::Exporters::Logger.new end

def create_measurement name:, value:, tags:
  measure = MeasureRegistry.get name
  return measure.create_measurement(value: value, tags: tags) if measure
  raise ArgumentError, "#{name} measure is not registered"
end
